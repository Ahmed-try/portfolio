PawCare

by Ahmed Younas — Mobile App Developer

🐾 Project Overview

PawCare is a Flutter-based mobile application designed for pet lovers. It integrates Firebase Authentication and an intelligent chatbot to help users detect dog breeds from images and receive real-time responses to their queries.

✨ Key Features

  • 🔐 Firebase Authentication for secure access
  • 🐶 Custom ML-powered dog breed detection
  • 💬 Smart AI chatbot with real-time replies
  • 🎨 Modern Flutter-based user interface
  • 🧠 Personalized suggestions and interactions
